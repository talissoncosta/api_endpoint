## Node.js end-point
### Steps to run the API

* In the root of the project run

`docker-compose up`

And now you are able to access the endpoint by the address:

`http://localhost:3001?name=John`

+ GET / - 
	+ PARAM
		name - string
		
To run with CURL:
 `curl -X GET   'http://localhost:3001/?name=John' `
